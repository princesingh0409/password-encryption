# Password Manager (Encryption)

A simple Python Password Manager that securely stores passwords using Fernet Encryption.

## Features

- Add Password
- View Password
- Password Encryption
- JSON Storage
- Secure Key Generation

## Technologies Used

- Python
- Cryptography
- JSON
- File Handling

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python password_manager.py
```
